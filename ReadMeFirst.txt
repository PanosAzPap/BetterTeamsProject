BetterTeams Project

Instructions:

1. Open the DBScripts folder
2. Open DBBetterTeams.sql script in MSSQL Express(NOTE: Set "Filename" path as your current path to mssqlexpress folder)
3. Execute the script
4. Open ProjectBetterTeams.sln
5. Go to Tools--> NuGet Package Manager--> Package Manager Console
6. In Package Manager Console Write "update-database" and press enter

You are ready to use BetterTeams!

Have Fun!!
